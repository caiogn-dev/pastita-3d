## 🔗 Relações Entre Projetos

```
PASTITA-3D (Customer)
    ↓ HTTP REST
PASTITA-DASH (Admin)
    ↓ HTTP REST
SERVER-MAIN (API)
    ├→ PostgreSQL (dados)
    ├→ Redis (cache + queue)
    └→ Serviços externos
```

**Fluxo Típico**:
1. Cliente usa PASTITA-3D
2. Admin usa PASTITA-DASH
3. Ambos se conectam a SERVER-MAIN
4. SERVER-MAIN integra com Mercado Pago, WhatsApp, etc
